Hello Proud User! Thank you for downloading Other Worlds Reboot mod, this is some useful information that might help you in the future:

> If you encounter any errors/bugs/oversights or any other type of shenanigans or crazyness, please report to the following link:
		https://forum.kerbalspaceprogram.com/index.php?/topic/164146-15-14-13-other_worlds-star-pack-reboot-redoing-the-whole-system-again

> You have downloaded this version from Github.

> If you want to install a new version of the mod, please remove the folder "OtherWorldsReboot" from the "GameData" folder before reinstalling.

> For the moment, no optional features are included.
